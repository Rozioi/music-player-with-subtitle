# 🎵 Music Player — React + Vite + React Icons

Проект из видео: создание плеера с текстом, как в Яндекс Музыке.

## 🚀 Стек технологий
- **React**
- **Vite**
- **React Icons**

## 📦 Установка и запуск

```bash
# Клонируем репозиторий
git clone https://github.com/Rozioi/music-player-with-subtitle.git
cd music-player-with-subtitle

# Устанавливаем зависимости
npm install

# Запускаем проект
npm run dev
